This repository contains the five required projects for the Responsive Web Design Certification from freeCodeCamp.

The projects are built using HTML and CSS and focus on:

- Modern layout techniques
- Responsive web design
- Web accessibility
- Building practical and user-friendly webpages

## Projects Included

- Build a Survey Form
- Build a Page of Playing Cards
- Build a Book Inventory App
- Build a Technical Documentation Page
- Build a Product Landing Page
